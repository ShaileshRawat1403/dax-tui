# DAX v0.2 Workstation UX

## Goal

Refine the DAX terminal workstation into a clearer AI execution control plane.

## Focus Areas

### 1. Stream Refinement

- Normalize event density
- Improve operator narration (e.g., "Explorer → scanning repository" instead of multiple lines)
- Better artifact emission visibility

**Example improvement:**

Before:

```
Explorer → reading README
Explorer → scanning modules
Explorer → scanning dependencies
```

After:

```
Explorer → scanning repository structure
  • README
  • modules
  • dependencies
```

### 2. Telemetry Panel

- Session state clarity
- Trust signal visibility
- Artifact counters
- Stage progress indicator

**Target format:**

```
Session
id: 18a4
workflow: repo-health

Stage
Executing (3/5)

Trust
0.55

Artifacts
4

Release
READY
```

### 3. Context Band

Add task summary above the stream:

```
Task
Analyze repository architecture

Workflow
repo-health

Mode
Normal
```

### 4. ELI12 Presentation

Simplify summaries when ELI12 mode is enabled:

Normal:

```
Verifier → trust score 0.55
release readiness → READY
```

ELI12:

```
The repository passed the safety checks.
It is considered ready for release.
```

### 5. Overlay Polish

- Consistent title headers
- Better ESC behavior
- Clearer footer hints

**Example overlay header:**

```
Artifact Detail — verification_report
```
