# DAX Positioning

This document freezes the product positioning for the current DAX release.

## Category

DAX is a mode-first terminal execution console for AI-assisted SDLC through natural language programming.

## Core Promise

DAX helps operators drive software delivery work in plain language while keeping execution visible, governed, and reviewable.

## One-Line USP

DAX is the terminal execution console for teams that want AI to do real work without losing control, context, or evidence.

## Expanded USP

DAX helps users plan, build, explore, document, and audit real software work in one terminal session with visible evidence and controlled execution.

## Market Gap

Most AI developer tools fall into one of three weak patterns:

- chat-first coding tools that answer well but govern execution poorly
- autonomy-heavy agent tools that obscure state and evidence
- dashboard-heavy developer platforms that add control surfaces but break terminal-native flow

DAX sits between those categories.

It is not a generic chat shell and not an opaque autonomous agent platform.

It is the operator console between the two.

## Differentiators

- Mode-first workflows: `plan`, `build`, `explore`, `docs`, `audit`
- Natural language programming for real SDLC work, not just code generation
- Governed execution with visible approvals, diffs, PM context, and audit signals
- Single-session continuity from intent to execution, review, and audit
- Terminal-native control without workstation or dashboard bloat

## Ideal User

DAX is strongest for:

- technical founders
- senior and staff engineers
- platform and infra teams
- AI-heavy product teams that need execution control
- operators shipping real repository changes under review pressure

## Positioning Guardrails

Do not position DAX as:

- a general-purpose chatbot
- a self-driving software engineer
- a broad enterprise governance suite
- a no-code tool for every audience

The strongest wedge is narrower:

DAX is for operators who want AI to do real software work in natural language without losing control of the session, the changes, or the evidence.
