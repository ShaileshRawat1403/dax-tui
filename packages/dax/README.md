# DAX

DAX is an AI-assisted execution agent framework, leveraging various AI models and tools to perform complex tasks. It aims to provide a deterministic, verifiable, and efficient way to interact with and automate software engineering workflows.

## Features

- **Agentic Execution**: Utilize AI models for task execution.
- **Tooling Integration**: Seamlessly integrate with various tools and APIs.
- **Deterministic Workflows**: Ensure verifiable and repeatable operations.
- **Extensible Architecture**: Easily add new AI models, tools, and workflows.

## Installation

To install DAX, clone the repository and install dependencies using `bun`:

```bash
git clone https://github.com/your-org/dax.git # Replace with actual repo URL
cd dax
bun install
```

## Usage

To run DAX, use the following command:

```bash
bun run dev
```

For type-checking, linting, and testing:

```bash
bun run typecheck
bun run lint
bun run test
```
